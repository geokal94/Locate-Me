# Locate-Me
Ionic 4 app to track geolocation of user using Google Maps API <br><br>
<h2>How to use</h2>
1) Clone this repository <br>
2) <code>npm install -g ionic cordova</code> if you don't already have ionic and cordova.<br>
3) <code>npm install</code> to install the required dependencies.<br>
4) Connect your Android device via USB and run <code>ionic cordova run android --device</code> 
to deploy the app to your Android device.<br>
5) If you want to run in the browser run <code>ionic serve</code>. 
However, facebook login is not available in the browser because cordova is not available.
